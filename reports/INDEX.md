# Results index

Navigation for all TRR result artifacts. **`RESULTS_TRR.md` is the master**
(complete, live); the others are focused outputs regenerated by `train/` modules
and `scripts/run_all.sh`.

| File | What it covers | Generator |
|---|---|---|
| [RESULTS_TRR.md](RESULTS_TRR.md) | **Master** — full empirical study: campaign, RAG-beyond-COVID, 7B-vs-32B, Graph-RAG, feasibility, operating points, limitations | (hand-curated + all runs) |
| [training_results.md](training_results.md) | Does training help? OOT + walk-forward; permutation importance | `python -m train.run` |
| [ablation_results.md](ablation_results.md) | Within-source fairness, calibration (Brier), precision@K | `python -m train.ablations` |
| [significance_results.md](significance_results.md) | Paired bootstrap CIs on RAG deltas (2/3 windows significant) | `python -m train.significance` |
| [threshold_results.md](threshold_results.md) | Alert-rate operating points + confusion matrices | `python -m train.threshold` |
| [backtest_results.md](backtest_results.md) | Economic de-risk overlay vs buy-and-hold | `python -m train.backtest` |
| [direction3_results.md](direction3_results.md) | 3-class up/flat/down (SIDEWAYS) — ~chance | `python -m train.direction3` |
| figures/*.png + stock_runs/fig_data.json | Reliability, backtest equity, campaign AUROC (interactive in webapp) | `python -m train.figures` |
| stock_runs/campaign/*.json | Raw per-window eval JSONs (32B, 7B, RAG, graphrag) | Kaggle kernels |

## Headline numbers
- Stock COVID crash AUROC **0.785** (+RAG **0.847**); broad 2016-2020 **0.710**.
- RAG beyond COVID: broad **+0.074** (p=0.009), bear **+0.065** (p=0.004).
- 7B + RAG **0.583** ≈ 32B baseline **0.570**; 32B+RAG **0.602** (best).
- Economic backtest: de-risk **+205% / −45% maxDD / 0.97 Sharpe** vs buy&hold +161% / −50% / 0.80.
- Negatives: Graph-RAG, 3-class direction, OHLCV volume, RAG-meta-feature — all no gain.
